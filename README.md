🌐 **Live site:** [perolahumana.org](https://perolahumana.org)

Institutional website developed for **Pérola Humana**, a non-profit organization focused on health, well-being and consciousness. The site presents the NGO's mission, services, events and educational content, with multi-language support and a fully responsive design 🌍.

## 🚀 Features

- 🌐 Multi-language support (PT, EN, FR, ES, AR) via Google Translate
- 📱 Fully responsive layout (mobile, tablet, desktop)
- 🎬 Health video section with embedded media
- 🗂️ Alphabetical disease index with expandable modals
- 🧘 Guided meditation audio player with progress bar
- 📖 Book showcase with drag-to-swipe carousel
- 🗓️ Events section
- 💌 Contact form
- ✨ Smooth animations powered by Framer Motion
- 🔤 Custom typography with Google Fonts (Cinzel, Rubik)

## 🛠️ Tech Stack

- ⚛️ Next.js 15 
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🎞️ Framer Motion
- 🧩 shadcn/ui
- 🔣 Lucide React
- 📊 Vercel Analytics
  

## 🌍 Multi-language

The site supports 5 languages using Google Translate integration. The language switcher is available both in the desktop navigation bar and in the mobile hamburger menu.
